# AIcleaner

AIcleaner 是一个 AI 驱动的文件盘点、归类和整理顾问桌面应用。

它的重点不是“直接删文件”，而是先帮你看清目录里有什么，再给出更稳妥的归类与整理路径：`盘点 -> 归类 -> 顾问`。

## 适合什么场景

- 下载目录、桌面、项目资料盘越来越乱，但你不想靠手工一个个翻。
- 你想先理解目录结构，再决定哪些内容该归档、重组或清理。
- 你希望 AI 参与整理，但不希望它一上来就直接批量删除。
- 你有一批混杂的文档、图片、压缩包、素材或项目文件，想先做结构化整理。

## 核心工作流

### 1. 盘点

盘点页负责选择目录、设置扫描范围、查看活动日志，并保存历史记录。

你可以在这里：

- 选择需要处理的目录
- 控制扫描深度和忽略路径
- 查看最近的盘点历史
- 把当前目录一键带入归类页或顾问页继续处理

盘点的目标是先建立目录上下文，而不是立刻执行整理动作。

### 2. 归类

归类页会结合模型能力，把文件按主题或用途组织成更清晰的结构。

你可以在这里：

- 选择模型和路由方式
- 控制批大小、摘要模式、排除项等整理参数
- 查看归类结果和执行日志
- 应用归类动作，并在需要时回滚最近一次结果

如果你已经知道自己想要一个更干净、更有层次的目录结构，通常可以直接从盘点进入归类。

### 3. 顾问

顾问页适合“我知道目录很乱，但还没想好怎么动手”的场景。

它会基于当前目录和历史上下文，帮你：

- 先看整体结构
- 给出自然语言建议
- 根据你的补充要求继续细化方案
- 在更明确的意图下生成可执行预览

如果你希望用对话方式推进整理，而不是自己调一堆参数，顾问页会更适合。

## 第一次使用前需要准备什么

首次使用前，先在左侧边栏点击 `服务商 API`，至少配置一个可用的大模型服务商。

当前界面内已经预置了多种常见服务商入口，例如：

- OpenAI
- DeepSeek
- Gemini
- Qwen
- GLM
- Kimi
- MiniMax

如果你希望归类阶段启用联网补充信息，还可以额外配置 Tavily；这不是必选项，没有它也可以完成本地目录盘点和大部分整理流程。

## 快速开始

### 1. 配置模型服务

点击左侧边栏的 `服务商 API`，填入至少一个可用的 API Key，并选择对应模型。

建议先只配一个你最常用的服务商，确认流程跑通后再扩展。

### 2. 从盘点开始

进入 `盘点` 页面：

- 选择要处理的目录
- 视情况调整扫描深度和忽略路径
- 启动盘点

盘点完成后，你可以在历史记录里再次加载结果，或者直接继续进入下一步。

### 3. 进入归类或顾问

盘点完成后有两条常见路线：

- 想直接整理结构：进入 `归类`
- 想先聊清楚怎么整理：进入 `顾问`

这两个页面都可以复用盘点结果，不需要每次都从头开始。

### 4. 需要时迁移缓存

如果你不希望应用数据继续放在当前默认位置，可以使用左侧边栏里的 `迁移缓存`。

这个入口直接嵌在工作流侧边栏里，不需要单独进入设置页。

## 典型使用方式

### 下载目录整理

先盘点整个下载目录，快速看出压缩包、安装包、图片、文档等主要堆积情况；再进入归类页，把杂乱内容拆成更清晰的分类结构。

### 项目资料归档

先对一个项目资料目录做盘点，再用顾问页讨论“哪些内容保留在主目录、哪些适合归档、哪些应分流到子目录”。

### 大目录先看后动

当你不确定一个目录里到底有什么时，先做盘点，再查看历史记录、结构概览和建议，而不是一开始就执行整理。

## 用户能直接感知到的能力

- 盘点历史可复用，不必每次重扫
- 归类结果支持执行与回滚
- 顾问会话支持围绕同一目录持续推进
- 界面支持中英文切换
- 缓存目录可以从侧边栏直接迁移

## 注意事项

- AIcleaner 当前仍处于早期迭代阶段，还不是一个面向生产环境的成熟产品。
- 它更适合“先理解、再整理”的工作流，不适合把它当成无脑批量删除工具。
- 某些能力依赖你配置的模型服务；没有可用 API 时，相关 AI 流程无法正常工作。
- 联网搜索属于可选能力，只有在你配置了对应搜索服务后才可用。
- 当前仓库内容和启动方式都是桌面端形态，不是可直接部署的 Web 应用。

## 最小开发入口

如果你只是想本地跑起来体验当前版本，可使用源码启动。

### 环境前提

- Node.js
- Rust
- Tauri 开发环境

### 启动方式

```bash
npm install
npm run tauri:dev
```

如需本地打包：

```bash
npm run tauri:build
```

当前仓库的脚本和打包配置主要面向桌面端工作流；如果你只是普通用户，阅读上面的产品说明和快速开始部分即可。
